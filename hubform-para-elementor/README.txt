Integre facilmente seus formulários Elementor com o Hubform.

== Description ==

O plugin Hubform para Elementor permite que você conecte seus formulários Elementor diretamente ao Hubform. Com esta integração, você pode enviar dados de formulários do Elementor para o seu Hubform de forma simples e eficiente.

Características principais:
* Integração fácil com o Elementor
* Envio direto de dados para o Hubform
* Configuração simples por formulário

== Installation ==

1. Faça upload da pasta do plugin para o diretório `/wp-content/plugins/`
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Ao criar ou editar um formulário no Elementor, você verá uma nova seção chamada "Hubform para Elementor"
4. Insira a URL do seu formulário Hubform no campo fornecido

== Frequently Asked Questions ==

= Este plugin é compatível com a versão mais recente do Elementor? =

Sim, o plugin é regularmente testado e atualizado para garantir compatibilidade com as versões mais recentes do Elementor e Elementor Pro.

= Como obtenho a URL do meu formulário no Hubform? =

Faça login na sua conta Hubform, crie ou selecione um formulário e copie a URL fornecida para esse formulário específico.
